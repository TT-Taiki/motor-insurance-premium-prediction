# Motor Insurance Pricing Model

This project aims to predict motor vehicle insurance premiums using regression-based machine learning models.

## Objective
- Predict net insurance premium amounts
- Compare baseline and regularized regression models
- Evaluate models using MAE

## Project Structure
- notebooks/: analysis and modeling notebooks
- data/: datasets (not tracked in Git)
- reports/: final report and presentation materials

## Tools
- Python, pandas, scikit-learn
- Jupyter Notebook
- GitHub, Jira, Notion
