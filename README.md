# motor-insurance-premium-prediction

This portfolio project predicts motor insurance premiums to support pricing decisions
by underwriting and pricing teams. The primary evaluation metric is MAE.

## Project structure
- `notebooks/` : EDA, feature engineering, modeling
- `docs/` : business brief, assumptions, decision log
- `slides/` : presentation materials
- `data/` : dataset notes (raw data is not shared)
